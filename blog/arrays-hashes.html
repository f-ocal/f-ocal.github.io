Array and Hashes

Array

As Ruby is completely object-oriented, arrays and hashes are the containers object to hold the collection of objects. An array's job is to store other objects including other arrays, hashes, classes, integers, strings and true and false .. any object. Arrays in Ruby are the storage that holds ordered lists of objects. They are written a single row of variable length. Ruby allows you retrieve information from an array, you can modify or even insert data to an array.

An array can be created in following ways:
  1. Array.new() method
    2. The method of new() is usual way, additionally it also can help to specify the size of the array. While indicating the size of the array, the contents of the array can be can be set.
    a = Array.new() ==> creates an empty array, return value will be [].
    a = Array.new(3) ==> allows you specify the size of the array to 3, all of three elements are set to nil.
    a = Array.new(4, "hello") ==> also allows you to initialize its contents. All of four elements of the array will be assigned to "hello"

  2. Literal Constructor
    3. Creating an array by using square brackets []
    a=[] ==> creating an array with square brackets.
    a= [1,2,"three", "a", "b",[5,3,10]] ==> you can also initialize the contents of the array within the brackets. You can nest another arrays within an array.
To access the elements of the array by referencing index numbers. The first element is positioned at "0", second element is at "1".. such as
a= [1,2,"three", "a", "b",[5,3,10]]
a=[2] ==> "three" and a=[5] ==> [5,3,10]

Hashes

Hash is a collection of objects like arrays except that each element contains a pair of "key and value". For instance, if we want to store the phone numbers of friends, than hashes would be the great way to store both the name of person and the number. Hashes also provide to lookup operations based on keys. Hashes also remember their key-insertion order and observe the order when you iterate over them.

Hashes can be created in following ways:
  1. Hash.new() method
    2. Creating an empty hash
    h = Hash.new() ==> creates an empty hash! When you ask for the value for non-existing key, it returs `nil`.
    h = Hash.new(0) ==> assigns hash default value to 0. When you call for a key that doesn't exist, it returns the default value of `0`.

  2. Literal Constructor { }
    3. Creating a hash by using curly braces { } ~very popular way~
    h={} ==> creating an empty hash.

  3. The Hash.[] and h = { "key" => "value"}
    4. Creating a hash by using square brackets, this method takes comma-separated list of items. !!!
    h.["New York", "NY", "San Francisco", "SF"] ==> This will mapped to a key/value pairs as per following
    h = {"New York" => "NY", "San Francisco" => "SF"} ==> This is also another way to create hashes by giving key and values!!!

  4. Defining keys with symbols
    5. Both following ways are quite similar to each other. You can either assign `key` after symbol `:` and followed by `=>` sign or you can assign `key` and follow with symbol `:`, and then assign them to `values` inside quotation marks.
    h=[:new_york => "NY", :year => "2015"] ==> This will generate a new hash by giving key/value pairs.
    h=[new_york: "NY", year: "2015"] ==> This is also another way to create hash by giving key/value pairs.
