Array and Hashes

Array

As Ruby is completely object-oriented, arrays and hashes are the containers object to hold the collection of objects. An array's job is to store other objects including other arrays, hashes, classes, integers, strings and true and false .. any object.

An array can be created in following ways:
  1. Array.new() method
    2. The method of new() is usual way, additionally it also can help to specify the size of the array. While indicating the size of the array, the contents of the array can be can be set.
    a = Array.new() ==> creates an empty array, the value returns to [].
    a = Array.new(3) ==> allows you specify the size of the array to 3, all elements are set to nil
    a = Array.new(4, "hello") ==> also allows you to initialize its contents. All the elements of the array is "hello"

  2. Literal Constructor
    3. Creating an array by using square brackets []
    a=[] ==> creating an array with square brackets.
    a= [1,2,"three", "a", "b",[5,3,10]] ==> you can also initialize the contents of the array within the brackets. You can nest another arrays within an array.
Arrays in Ruby are the storage that holds ordered lists of objects. They are written a single row of variable length. Ruby allows you retrieve information from an array, you can modify or even insert data to an array. You can access elements within the array by referencing the index numbers. The first element is positioned at "0", second element is at "1"... as per below example:
a= [1,2,"three", "a", "b",[5,3,10]]
a=[0] ==> 1
a=[2] ==> "three"
